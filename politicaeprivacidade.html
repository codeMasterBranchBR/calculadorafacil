Política de Privacidade

Última atualização: 07/03/2025

1. Introdução

Bem-vindo à nossa Política de Privacidade. Esta página explica como coletamos, usamos e protegemos as informações dos usuários do nosso aplicativo de calculadora.

2. Coleta de Dados

Nosso aplicativo não coleta nenhum dado pessoal dos usuários. Todas as operações são realizadas localmente no dispositivo.

3. Permissões

O aplicativo pode solicitar permissões básicas, como acesso ao armazenamento, apenas para salvar cálculos (se aplicável). Nenhuma informação é enviada para servidores externos.

4. Publicidade

Nosso aplicativo pode exibir anúncios, que podem coletar dados anônimos para fins de personalização. Consulte a política de privacidade dos provedores de anúncios para mais informações.

5. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Recomendamos que os usuários revisem esta página regularmente para se manterem informados.

6. Contato

Se tiver dúvidas sobre esta Política de Privacidade, entre em contato conosco pelo e-mail: faturamaenterprise@gmail.com.
